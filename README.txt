---To Run---
1. Pull all contents into a folder
2. Open command prompt and change directory to just inside this folder
3. Run server.bat
4. Press listen button on server GUI
5. Open new command prompt and change directory to just inside folder.
6. Run client.bat
7. Enter the IP address of the computer running the server,
   and enter 8000 as the port.
8. Repeat steps 5-7 for another client.
9. Play the game and enjoy!
